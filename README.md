# Responsive Landing Page

##  Task 01 – SkillCraft Technology

This project is a responsive landing page featuring a fixed navigation bar with interactive hover effects and dynamic style changes on scroll.

##  Features
- Fixed navigation menu visible on all pages
- Navbar style changes when scrolling
- Hover effects on menu items
- Fully responsive design for desktop, tablet, and mobile
- Clean UI using HTML, CSS, and JavaScript

##  Technologies Used
- HTML5  
- CSS3  
- JavaScript (DOM manipulation & event handling)

##  Project Structure
├── index.html
├── style.css
└── script.js

##  How to Run
1. Download or clone the repository
2. Open `index.html` in any modern web browser
3. Scroll the page to see navbar interaction

##  Objective
To practice responsive web design, fixed navigation layouts, hover interactions, and scroll-based DOM manipulation using vanilla JavaScript.
